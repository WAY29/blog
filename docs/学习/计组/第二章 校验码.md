# 校验码
## 奇偶校验码
校验位加有效信息中的1为奇数则是奇校验
校验位加有效信息中的1为偶数则是偶校验
奇偶校验均不能发现偶数位错，也无法定位错误

## 海明校验码
海明码设计思路：将信息位分组进行**偶校验**->多个信息位->多个校验位标注出错位置
### 偶校验
若分组中存在偶数个1，则校验码为0，否则校验码为1
